# React / TypeScript / Next.js porfolio

This is a mobile responsive portfolio built with React TypeScript and Next.js 

### Getting Started

Install all dependencies:
npm install 

second, run the development server:

```bash

npm run dev
# or
yarn dev
# or
pnpm dev
```
